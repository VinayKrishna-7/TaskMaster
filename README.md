# 🚀 TaskMaster Pro V2

TaskMaster Pro V2 is a modern productivity-focused task management web application designed to help users organize tasks, track progress, monitor productivity, and stay focused on their goals.

Built with pure HTML, CSS, and JavaScript, the application provides a clean UI, responsive design, dark mode support, analytics, and persistent local storage.

---

## ✨ Features

### 📋 Task Management
- Create tasks with:
  - Title
  - Description
  - Category
  - Priority Level
  - Due Date
- Edit existing tasks
- Delete tasks
- Mark tasks as completed

### 🔍 Smart Search & Filtering
- Search tasks instantly
- Filter by:
  - Active Tasks
  - Completed Tasks
  - High Priority Tasks
- Filter by category

### 📊 Productivity Dashboard
- Total Tasks Counter
- Completed Tasks Counter
- Pending Tasks Counter
- Productivity Score
- Completion Rate
- Daily Streak Tracking

### 🎯 Focus Section
- Displays high-priority pending tasks
- Helps users focus on important work

### 📅 Calendar View
- View upcoming deadlines
- Track scheduled tasks
- Monitor upcoming events

### 📈 Analytics
- Completion Rate
- Overdue Tasks
- Category Usage Statistics

### 🌙 Dark Mode
- Light/Dark theme toggle
- Theme preference saved automatically

### 💾 Local Storage Support
- Tasks persist after page refresh
- Theme preference saved
- Productivity streak saved

### 📦 Backup & Export
- Export tasks as JSON backup
- Restore data manually if required

### ⌨️ Keyboard Shortcuts
| Shortcut | Action |
|-----------|---------|
| Ctrl + K | Focus Search |
| ESC | Close Task Modal |
